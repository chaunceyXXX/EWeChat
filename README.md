# 企业微信自动发送助手

这是一个帮助你自动将本地文件发送到企业微信的小工具。

## 🚀 快速开始

1.  **启动服务**:
    *   后端: `venv/bin/python backend/main.py`
    *   前端: `npm run dev`
2.  **打开页面**: 访问 http://localhost:5173

## 📝 企业微信配置指南

为了让软件能够发送消息，你需要登录 [企业微信管理后台](https://work.weixin.qq.com/wework_admin/loginpage_wx) 获取以下信息：

### 1. 企业ID (CorpID)
*   **位置**: `我的企业` -> `企业信息` -> 最底部的 `企业ID`。
*   **说明**: 类似于企业的身份证号，以 `ww` 开头。

### 2. 创建自建应用 (获取 AgentID 和 Secret)
*   **位置**: `应用管理` -> `应用` -> `自建` -> 点击 `+ 创建应用`。
*   **步骤**:
    1.  上传一个 Logo，起个名字（比如“文件小助手”）。
    2.  可见范围：选择你可以发送消息的人员或部门。
    3.  创建完成后，进入应用详情页。
*   **AgentID**: 应用详情页顶部的 `AgentId`（数字）。
*   **Secret**: 应用详情页的 `Secret` -> 点击 `查看` -> 需要手机企业微信扫码确认 -> 获取一串长字符。

### 3. 接收对象 (UserID / PartyID)
*   **接收用户 (UserID)**:
    *   **位置**: `通讯录` -> 点击具体成员 -> `账号` 字段。
    *   **说明**: 如果想发给所有人，可以填 `@all`。如果有多个人，用竖线 `|` 分隔，例如 `zhangsan|lisi`。
*   **接收部门 (PartyID)**:
    *   **位置**: `通讯录` -> 点击部门名称右侧的三个点 -> `部门ID`。
    *   **说明**: 如果填了 UserID，这个可以留空。

## 📁 监控文件夹配置
*   请填写**绝对路径**。
*   macOS 示例: `/Users/yourname/Documents/Reports`
*   Windows 示例: `D:\Data\DailyReports`

## ⚠️ 常见问题
*   **报错 "ip不可信"**: 企业微信为了安全，第一次使用 API 时可能会报错。你需要根据报错信息中的 IP 地址，去 `应用管理` -> `你的应用` -> `企业可信IP` 中配置一下这个 IP。
